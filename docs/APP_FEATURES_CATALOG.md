# App features catalog

The **master feature list** for every portfolio app lives in the dashboard source so it stays in sync with deployments:

1. Open **`BVS_Dashboard_v2_1.html`** on your site (e.g. `/BVS_Dashboard_v2_1.html`).
2. Go to the **📋 Features** tab.
3. **Print / Save as PDF** — use the browser print dialog and choose *Save as PDF*.
4. **Download .md** — exports the same content as Markdown for GitHub or docs.

To add or change bullets, edit the `APP_FEATURE_CATALOG` object in `BVS_Dashboard_v2_1.html` (keys match each app’s `id` in `APPS`).
